Java教程第一课

2022-04-09 1 min read

Java，这个高级编程语言一直在编程语言中地位很高，但是非常难学o(╥﹏╥)o
所以，我今天把我学Java的经验分享给你们。

一、输出

上代码

Java 17

public class PrintJava {
    public static void main(String[] args) {
        System.out.println("Java")
    }
}

我们来一行一行看一下
首先第一行中 public 是权限修饰符，作用是声明类 PrintJava 是公共的
class、static是关键字，声明它们的类型，void 是一个数据类型，声明 main 的返回类型

在Java中，只能存在一个 public class，这个 class 的名字就是文件名

main 是一个函数，其中的参数 String[]和 args在后面会学到
函数的内容就是System.out.println("Java")，意思是输出字符串"Java"
接下来我们将文件保存为PrintJava.java（注意一定要注意大小写）
接下来运行

javac PrintJava.java
java PrintJava
